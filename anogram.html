Anogram

Android · Bluetooth LE · offline messaging

Anogram is an Android messaging app built around a simple idea: two nearby phones should be able to chat even when there is no internet, account, or server involved.

How it works

The app uses Bluetooth Low Energy for discovery and message exchange. Each device can advertise a service, scan for nearby peers, connect, exchange messages, and store chat data locally.

Why it was useful to build

Anogram is not one of my main AI/ML projects, but it taught me a different kind of engineering: dealing with Android APIs, hardware behavior, connection timing, missed advertisements, and state that changes outside the app’s control.

What I learned

  • BLE looks simple on paper, but real devices make it messy.
  • Clean architecture helps keep app logic readable when the data layer is hardware-specific.
  • Offline-first systems force you to think carefully about state and failure modes.
